# Code of Conduct

This project is committed to a respectful and inclusive environment.

## Expected behavior

- Be kind and constructive.
- Assume good intent.
- Focus on the work and the problem, not the person.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Posting sensitive information without consent.
- Deliberately disrupting discussion or collaboration.

## Enforcement

Reports should be sent to the maintainers. Inappropriate behavior may result in removal from the project.
